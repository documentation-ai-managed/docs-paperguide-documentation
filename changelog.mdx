<Update label="2024-09-20" description="v2.0.0" tags={["feature", "breaking"]}>

## Breaking Changes

- Updated authentication to use JWT tokens instead of API keys. Migrate by regenerating tokens in your settings.

<CodeGroup tabs="JavaScript,Python">
```javascript
// Old
const response = await fetch('/api/docs', {
  headers: { 'API-Key': 'old-key' }
});

// New
const token = 'your-jwt-token';
const response = await fetch('/api/docs', {
  headers: { 'Authorization': `Bearer ${token}` }
});
```
```python
# Old
response = requests.get('/api/docs', headers={'API-Key': 'old-key'})

# New
token = 'your-jwt-token'
response = requests.get('/api/docs', headers={'Authorization': f'Bearer {token}'})
```
</CodeGroup>
