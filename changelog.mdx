## Release History

- Updated UI themes for better accessibility compliance.

- Integrated webhook support for custom notifications.
- Added version comparison tool for diff reviews between commits.

## Improvements

- Optimized search indexing to handle 10x more content efficiently.
- Streamlined invitation workflow with bulk import from CSV.

</Update>
